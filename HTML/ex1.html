A Linguagem HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas da Web. Documentos HTML podem ser interpretados por navegadores.

Até este momento, vimos os seguintes recursos de linguagem:

Recurso Tags associadas Descrição
Links <a> Criar links entre as páginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar listas de itens não ordenados
Imagens <img> Inserir imagens
